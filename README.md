# lvspline
Parametrized Left Ventricle Geometry and Mesh Generator (backend: Gmsh)

Usage: 

  - python create_mesh.py
  
Output: Files generated by Gmsh

  - out_lv_mesh.geo (geometry) 
  - out_lv_mesh.msh (finite element mesh)
  
Visualization:

  - gmsh out_lv_mesh.msh
